# AmyBX Computer Vision Lab

AmyBX Computer Vision Lab is a computer vision research group focusing on small object detection, object tracking, camouflaged object detection, and related visual perception tasks.

## Research Directions

### Small Object Detection

We study robust and efficient methods for detecting small objects in challenging scenarios, including infrared small target detection and dense small object detection.

### Object Tracking  

We focus on visual object tracking, including single object tracking, robust tracking in complex scenes, and tracking-related visual perception tasks.

### Camouflaged Object Detection

We investigate methods for detecting and segmenting camouflaged objects by exploring texture, boundary, context, and multi-scale visual cues.
